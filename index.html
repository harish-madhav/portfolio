Graph Coloring Game

Flask JavaScript HTML/CSS

The Graph Coloring Game is an interactive web-based puzzle game based on 4 color theorem in graph theory where players are tasked with coloring a graph's nodes, ensuring that no two adjacent nodes share the same color. The game challenges the player's problem-solving skills and knowledge of graph theory concepts, all while being fun and educational.
